Déploiement sur Render : configuration Docker en 3 lignes

render.yaml

services: - type: web name: full-video-to-clips-v3 runtime: docker envVars: - key: PORT value: 10000

Le fichier render.yaml configure le déploiement automatique.

Détails de l'infrastructure

Dockerfile optimisé
Installe Python 3.11, FFmpeg, et les dépendances Python via requirements.txt.
Gunicorn
Le serveur WSGI écoute sur le port 10000 (injecté par Render via la variable d'environnement $PORT).
Reproductibilité
Le déploiement est entièrement reproductible grâce à la conteneurisation, évitant les problèmes de dépendances liés à FFmpeg.
