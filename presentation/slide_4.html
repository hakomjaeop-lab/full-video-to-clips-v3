Architecture technique : Flask + FFmpeg + Docker

Le projet repose sur une stack légère et efficace :

Flask
Framework web Python pour gérer les routes HTTP et le rendu HTML.
FFmpeg / ffprobe
Moteur de traitement vidéo (découpage, recadrage, encodage).
Gunicorn
Serveur WSGI de production pour exécuter l'application Flask.
Docker
Conteneurisation pour un environnement d'exécution reproductible.
Render
Plateforme cloud de déploiement via le fichier de configuration render.yaml.
